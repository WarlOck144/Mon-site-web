Technologies utilisés

Pour la construction du site, les technologies suivantes seront employées :


  • HTML (HyperText Markup Language): Structure de base du contenu sur chaque page.

  • CSS (Cascading Style Sheets) : Stylisation des éléments, ajoutant de la couleur, des polices spécifiques, et un agencement esthétique.

Éléments multimédia

Pour rendre le site plus interactif et intéressant, j’intègrerai des éléments multimédias :


  • Images : Photos ou illustrations représentant mes centres d'intérêt et activités extra-scolaires, optimisées pour le web pour un chargement rapide.

  • Vidéos : Une courte vidéo représentative pour illustrer un centre d'intérêt dans la page "About me".

Accessibilité et ergonomie

Pour garantir une bonne accessibilité :


  • Responsive Design : Le site sera conçu pour s’adapter aux différents écrans (ordinateurs, tablettes, téléphones) afin d’offrir une expérience utilisateur optimale sur chaque appareil.

  • Contraste et Taille de Police : Les éléments importants auront un bon contraste et la taille des polices sera optimisée pour faciliter la lecture et rendre le contenu accessible à tous.

  • Balises HTML Sémantiques : Des balises HTML sémantiques telles que <header>, <nav>, <section>, et <footer> seront utilisées pour organiser le contenu de manière logique. Cela améliore l’accessibilité, notamment pour les utilisateurs d’outils d’assistance comme les lecteurs d’écran.
